# Course 03 – Supporting AWS Workloads

## Overview
Operational support and troubleshooting for AWS workloads.

## Topics Covered
- Monitoring with CloudWatch
- Logging strategies
- Troubleshooting EC2 and networking issues
- AWS Support tools

## Practical Activities
- Setting up CloudWatch alarms
- Diagnosing connectivity problems
- Reviewing logs for performance issues

## Outcome
Capability to maintain and troubleshoot production cloud environments.
